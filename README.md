# starbucksproject
[[JSP 스타벅스 프로젝트]]

기획의도 : 스타벅스 페이지 클론 코딩을 통한 JSP 구조 실습

기간 : 2022.10.19 ~ 2022.10.26

개발 인원 : 8명

개발 플랫폼 : Window10

개발 툴 : draw.io, exERD, EXCEL, SQL Developer, Eclipse

사용언어 : JAVA, SQL, HTML, CSS, JavaScript

개발 기술 : JDBC, JQuery, JSP

담당 업무 : 팀원, 스타벅스 메인페이지, 뉴스 게시판, 매장별 이벤트 게시판 페이지 생성

코드 구현 : JSP MVC 패턴 2를 활용한 웹 

기능 : 요구분석 -> 논리모델링 -> 물리모델링 -> DB 데이터 구성 -> 테이블 생성 -> JSP MVC 패턴 코드 작성 -> 실제 페이지 구성
